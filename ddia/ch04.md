##### JSON, XML and Binary Variants

JSON, XML, CSV 的问题:

- 数字处理
  - XML 和 CSV 无法分辨数字和由数组成的字符串
  - JSON 在处理超过 2^53 的整数时有精度问题 (JSON 中的所有数字都是 float 64)
- JSON 和 XML 不支持二进制数据
- CSV 没法很好地处理复杂的数据结构和结构变化, 且处理转义的规则十分模糊
